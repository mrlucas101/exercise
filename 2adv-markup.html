    Links

    Links use the tag and display inline. They must have an href attribute and can use target to open in a new tab

    Links have different pseudo-class states, if they're hovered or have been visited, for example.

    Images

    Images use the img tag, which is self-closing, and they might be inside a figure tag. Images use the src= attribute to point to the source, which could be anywhere. Images display inline by default.
